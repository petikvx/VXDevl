INTRODUCTION
------------------------------------------------------------------------------

 All entries on this list are formated as shown below. We use the CARO naming 
 convention for macro viruses:

  ---------------------------------------------------------------------------
  *ru+ Type://Platform/Strain.Variant.Addinfo:LC   =Aliases    Macintosh-Name
  ---------------------------------------------------------------------------
 
  *        = In-The-Wild (ITW) as mentioned on Joe Wells's "In-The-Wild List"
  r        = This virus has been renamed (e.g. suffix added)
  u        = Updated; this is only being used to correct errors from previous
             versions of the _same_ month (like spelling mistakes, etc.)
  +        = This virus is new since last edition
  Type     = virus, intended, trojan, joke, generator, dropper
  Platform = Word2Macro (W2M), WordMacro (WM; Word versions 5, 6 & 7),
             Word97Macro (W97M), ExcelMacro (XM; pre-Excel97-versions),
             ExcelFormula (XF), Office97Macro (O97M), OfficeMacro (OM),
             AccessMacro (AM), Access97Macro (A97M), Access2Macro (A2M),
             AmiMacro (Lotus AmiPro/WordPro), LotusMacro (Lotus 1-2-3)
  Strain   = Family name of the virus, trojan, etc.
  Variant  = letter code (e.g. A, B, AA, ...) followed optionally by an 
             evolution phase (e.g. A1, A2, ...)
  Addinfo  = additional information
  LC       = Language Code, platform for special language;
             En (English: default), Tw (Chinese/Taiwan), NL (Dutch), 
             De (German), It (Italian), Fr (French), Jp (Japanese)
             Es (Spanish), Br (Brazilian)

 A special notation applies to generators:

      -------------------------------------------
      generator://Platform/Strain_Variant_Version
      -------------------------------------------

 where:
  Variant  = number (e.g. generator://WordMacro/NJ-WMVCK1)
  Version  = letter (e.g. generator://WordMacro/NJ-WMDLK1A)
             (this is used only, when different versions for different 
             operating systems exist (like Macintosh, Windows 16bit, 
             Windows 32bit,...)

 The viruses are named by using the generator's name (but skiping the
 "Version" field) and appending a ".<letter>" (E.g. NJ-WMVCK2.A
 is one virus generated by NJ-WMVCK2, NJ-WMVCK2.B is another one;
 NJ-WMDLK1.A would be one generated by all of NJ-WMDLK1A to NJ-WMDLK1E)

 Remark:  Some AV products may assign names in conflict with systematic 
          CARO names. Such names are put in parantheses in column "Alias".

AVAILABILITY
==============================================================================

 This list is published quarterly and can be downloaded via FTP from the 
 following site:

 ftp://agn-www.informatik.uni-hamburg.de/pub/texts/macro/

 The filenames used are:
    MACROLST.yym        (long version)
    MACROL_S.yym        (short version)
    NEWMACLS.yym        (list of new malware)

 "yym" stands for:

    yy = Year,
    m  = Month ("1"..."9" for January...September and
                "A" = October, "B" = November, "C" = December.
